Frontend Registration

## Prerequisite
1. Node JS Stable Version

## Running Development
### Package installation
```shell
npm install
```
### Local Setup
1. Create an `.env` file.
2. Create an entry in `.env`, `VITE_API_BASE_URL=http://localhost:8000`

### Local
Run it locally on port `5173`
```shell
npm run dev
```
